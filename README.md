## 📖 About the Project

Gastropods perform crucial ecological functions in intertidal zones and serve as a valuable food source rich in iodine and protein. However, the taxonomic classification of gastropods is highly complex due to cryptic diversity and immense morphological variation, leading to a significant taxonomic impediment. Manual identification is laborious, time-consuming, and requires specialized expertise.

This project introduces a deep learning-based **Gastropod Classification System** utilizing state-of-the-art computer vision techniques to automate morphological feature extraction and taxonomic classification. By leveraging the **YOLOv8 Instance Segmentation** architecture, this system can accurately identify 23 distinct species of marine gastropods down to their complex morphological boundaries.
